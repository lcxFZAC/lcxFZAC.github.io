Python基础用语

字符串

字符串就是多个字符的集合
由双引号””或者单引号’’包围
字符串可以包含英文、数字、中文及各种符号

print输出字符串格式如下：
print(“内容”)或print(‘内容’)
字符串要放在小括号( )中传递给 print
让 print 把字符串显示到屏幕上
这种写法在 Python 中被称为函数（Function）

注意

输入符号时必须是英文半角状态
Python严格区分大小写
print和Print是不同的
同一print函数里的字符串之间不会自动换行
需要用\n进行换行
python不需要再语句结尾加上分号，但也可以加

数字

print除字符串也能输出数字
将数字或表达式直接放入print语句即可输出
输出时不能用引号包围，否则将输出字符串
数字配合运算符号输出可输出运算结果
不能单独将多个数字同时放在一个函数中

数字类型

数字类型分为：整型、浮点型、布尔型、复数类型

运算符及意义

基础运算符

“+”—–两个数字相加，或者是字符串连接
“-“—–两数字相减
“*”—–两数字相乘，或者是或是返回一个重复若干次的字符串
“/“—–两个数相除，结果为浮点数（小数）
“//“—–两个数相除(整除)，结果为向下取整的整数
“%”—–取模，返回两个数相除的余数
“**”—–幂运算，返回乘方结果

比较（关系）运算符

“==”—–比较两个对象是否相等
“!=”—–比较两个对象是否不相等
“>”—–大小比较，例如 x大于y 将比较 x 和 y 的大小，如 x 比 y 大，返回 True，否则返回 False
“<”—–大小比较，例如 x小于y 将比较 x 和 y 的大小，如 x 比 y 小，返回T rue，否则返回 False
“>=”—–大小比较，例如 x>=y 将比较 x 和 y 的大小，如 x 大于等于 y，返回 True，否则返回 False
“<=”—–大小比较，例如 x<=y 将比较 x 和 y 的大小，如 x 小于等于 y，返回 True，否则返回 False

其余运算符暂时未学，如有需要自行百度（滑稽）

IDE（集成开发环境）

IDE 是 Integrated Development Environment 的缩写，中文称为集成开发环境，用来表示辅助程序员开发的应用软件，是它们的一个总称。

运行C语言必须有编译器
运行python必须有解释器
实际开发还需要各种辅助工具
这些工具集合在一起，统称IDE（集成开发环境）

函数（FunctionType）

与类和实例无绑定关系的function都属于函数（function）

方法（MethodType）

与类和实例有绑定关系的function都属于方法（method）

方法是只能依靠类或者对象来调用的，有针对性

库

库的引用
import语句接库名可以进行库的引用

turtle库 和 random库

由于只学习了一部分知识，故此文只讲这两个库

turtle（海龟）库

turtle库常用于绘制图形
turtle.属性()可进行库的调用
其中
penup—抬笔
pendown—落笔
pencolor—墨色
pensize—笔头大小
fd(forward)—直线
radius—弧度
left—向左转（后接旋转角度）
right—向右转（后接旋转角度）
fillcolor—填充颜色
beginfill—开始填充（结束填充同理）

random库(只有一丢丢知识)

random用于取随机数
用random.randint(a,b)可以生成一个[a , b]之间的整数
用randrange(m , n ,[ k ])可以生成[m , n)之间以k为步长的随机整数
用uniform(a,b)可以生成一个[a , b]之间的随机小数
